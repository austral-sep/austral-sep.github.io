使用 color 的标签
<font color="red">红色</font>
红色 : color = “rgb(200, 100, 100)”
绿色 : color = “green”
蓝色 : color = “#007bff”

使用 style 的标签
<font style="color: red">红色</font>
黄色 : style = “color: #ddff00”
使用 rgb 颜色值 : style = “background: #c86464”
