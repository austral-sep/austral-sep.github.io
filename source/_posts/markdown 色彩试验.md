使用 `color` 的标签
`<font color="red">红色</font>`
<font color = "rgb(200, 100, 100)">红色</font> : color = "rgb(200, 100, 100)"
<font color="green">绿色</font> : color = "green"
<font color="#007bff">蓝色</font> : color = "#007bff"

使用 `style` 的标签
`<font style="color: red">红色</font>`
<font style="color: #ddff00">黄色</font> : style = "color: #ddff00"
<font style="background: #c86464">使用 rgb 颜色值</font> : style = "background: #c86464"
